## Rock–Paper–Scissors (Python Version)

A simple Rock–Paper–Scissors game built entirely with Python.
This was the original version of the project before integrating HTML, CSS, JavaScript, or a leaderboard system.

---

## Features
-Play Rock, Paper, Scissors against the computer.
-Randomized computer choices.

---

## Technologies Used
Python
