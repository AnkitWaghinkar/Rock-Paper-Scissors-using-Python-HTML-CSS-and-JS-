## Rock–Paper–Scissors (Python Version)

A simple Rock–Paper–Scissors game built entirely with Python.
This was the original version of the project before integrating HTML, CSS, JavaScript, or a leaderboard system.

---

## Features
-Play Rock, Paper, Scissors against the computer.
-Randomized computer choices.

---

## Technologies Used
Python

---

## How game works
-The player chooses one option: rock, paper or scissors.
-The computer randomly selects its move.
-The program compares both moves and displays the result:
  •Player wins
  •Computer wins
  •Tie

---

## Improvements
-Added a leaderboard system
-Converted to a web-based UI using HTML, CSS, JS
-Score stored in database.
-Added lottie animations for win, lose, tie & audio for win, lose, tie and mouse click.
